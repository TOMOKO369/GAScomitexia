# 在庫管理システム (Stock Sync) - Googleフォーム入力・Web表示版

スマートフォン等の **Googleフォーム** から入出庫を入力し、**洗練されたWebダッシュボード (index.html)** でリアルタイムに在庫を確認できるシステムです。

## 📁 ファイル構成
- `index.html` : 在庫状況を確認するためのWebダッシュボード画面です。GitHub Pages等で公開します。
- `Code.gs` : Google Apps Script (GAS) に貼り付けるバックエンドコードです。

---

## 🚀 設定手順

### ステップ1：Googleフォームとスプレッドシートの準備
1. Googleフォームを新規作成し、以下の3つの質問を作成します。
   - **品目** （記述式 または プルダウン）
   - **入庫数** （数値のみ入力可能にする）
   - **出庫数** （数値のみ入力可能にする）
   *(※質問のタイトルは必ず「品目」「入庫数」「出庫数」としてください)*
2. フォームの「回答」タブを開き、「スプレッドシートにリンク」をクリックして新しいスプレッドシートを作成します。

### ステップ2：GASコードの貼り付け
1. 開いたスプレッドシートのメニューから「拡張機能」＞「Apps Script」をクリックしてエディタを開きます。
2. 最初から入っているコードをすべて消し、このフォルダにある `Code.gs` の内容をコピーして貼り付け、保存（Ctrl+S / Cmd+S）します。

### ステップ3：トリガーの設定（自動計算・自動メール用）
1. GASエディタの左メニューにある時計アイコン（「トリガー」）をクリックします。
2. 右下の「トリガーを追加」ボタンを押し、以下の**2つのトリガー**を作成します。

**【① フォーム送信時の自動計算トリガー】**
- 実行する関数: `onFormSubmit`
- イベントのソース: `スプレッドシートから`
- イベントの種類: `フォーム送信時`
- （保存してアクセスを許可してください）

**【② 毎朝の自動チェックトリガー】**
- 実行する関数: `checkStockDaily`
- イベントのソース: `時間主導型`
- 時間ベースのトリガーのタイプ: `日付ベースのタイマー`
- 時刻を選択: `午前 8時～9時`（お好きな時間帯）
- （保存）

### ステップ4：GASをWebアプリとして公開（表示用データの提供）
1. GASエディタ右上の青い「デプロイ」ボタンを押し、「新しいデプロイ」を選択します。
2. 歯車アイコンから「ウェブアプリ」を選びます。
3. 以下の通り設定して「デプロイ」を押します。
   - **実行するユーザー**: `自分`
   - **アクセスできるユーザー**: `全員`
4. 表示された **「ウェブアプリのURL」をコピー** します。

### ステップ5：ダッシュボード画面 (index.html) の設定と公開
1. このフォルダ内の `index.html` をエディタ（VSCodeやメモ帳）で開きます。
2. 下部にある `<script>` タグ内の以下の行を探します：
   ```javascript
   const GAS_WEB_APP_URL = 'YOUR_GAS_WEB_APP_URL_HERE';
   ```
3. `'YOUR_GAS_WEB_APP_URL_HERE'` の部分を、**ステップ4でコピーしたURL** に置き換えて保存します。
4. この `index.html` をご自身の GitHub のリポジトリにアップロードし、`Settings > Pages` から `main` ブランチを指定して保存します。
5. 数分後、発行された `https://ユーザー名.github.io/リポジトリ名/` にアクセスすると、現在の在庫状況が一覧でかっこよく表示されるダッシュボードが完成します！

### 💡 使い方
- スマホからは **Googleフォーム** を開いて、入出庫のデータを入力・送信します。
- 在庫の確認は、GitHub Pagesで公開した **ダッシュボードURL** にアクセスして確認します。（在庫が2個以下のものは赤く警告表示されます）
